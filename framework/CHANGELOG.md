# Changelog — ReSCI Framework

Le format s'inspire de [Keep a Changelog](https://keepachangelog.com/fr/).

## [v11] — 2026-01

### Ajouté
- Ajout automatique des ports web courants (80/443/8080/8443) au scan
- AD Full auto-activé quand Kerberos (port 88) est détecté avec des credentials
- Loot mode auto-activé lorsque des services web sont trouvés
- Export Markdown (`--report-level summary|full`) et export JSON Notion
- Mode VPN (`--vpn-mode`) optimisé pour HTB
- Validation de la cible (IP / hostname RFC 1123)
